Kapsamlı Mobil Tıbbi Terimler Sözlüğü

1. PROJE TANIMI

Pharmadict, eczacılık öğrencileri ve sağlık profesyonelleri için geliştirilmiş kapsamlı bir mobil tıbbi terimler sözlüğü uygulamasıdır.

Hedef Kitle

  • Eczacılık öğrencileri
  • Tıp öğrencileri
  • Sağlık profesyonelleri
  • Araştırmacılar

Temel Amaç

İlaçlar, bitkiler, vitaminler, mineraller, hastalıklar, böcekler, bileşenler ve anatomi terimlerini içeren zengin bir veritabanı sunarak tıbbi eğitimi desteklemek.

12.1 Terim ID Oluşturma

const generateId = (category: string, latinName: string): string => {
  return `${category}_${latinName.toLowerCase().replace(/\s+/g, "_")}`;
};
// Örnek: "İlaçlar_aspirin"

12.2 Arama Algoritması

const searchTerms = (query: string): SearchResult[] => {
  const queryLower = query.toLowerCase();
  return terms
    .filter(
      (term) =>
        term.latinName.toLowerCase().includes(queryLower) ||
        term.turkishName.toLowerCase().includes(queryLower) ||
        term.definition.toLowerCase().includes(queryLower)
    )
    .map((term) => ({ term, score: calculateRelevance(term, query) }))
    .sort((a, b) => b.score - a.score);
};

12.3 Favori Toggle

const toggleBookmark = async (termId: string): Promise<void> => {
  const bookmarks = await AsyncStorage.getItem("bookmarked_terms");
  const bookmarkList = bookmarks ? JSON.parse(bookmarks) : [];

  const index = bookmarkList.indexOf(termId);
  if (index > -1) {
    bookmarkList.splice(index, 1); // Kaldır
  } else {
    bookmarkList.push(termId); // Ekle
  }

  await AsyncStorage.setItem("bookmarked_terms", JSON.stringify(bookmarkList));
};
